# Eleccions Generals

**Eleccions Generals** is a database project for processing, storing, and querying Spanish general election results. It reads official fixed-width `.DAT` files published by Spain's Instituto Nacional de Estadística (INE), imports them into a relational MySQL database, and provides a structured SQL query library for electoral analysis.

## Data coverage

The project covers the **2016 Spanish General Elections** (`02201606` data series), with data structured at three geographic levels:

- **Comunitats Autònomes** — 17 autonomous communities
- **Provincies** — 52 provinces with seat allocations
- **Municipis** — all municipalities with polling station counts and vote breakdowns

## Database schema overview

The MySQL database (`mydb`) contains 11 tables that model the complete electoral data hierarchy:

| Table | Description |
|---|---|
| `comunitats_autonomes` | Spain's 17 autonomous communities |
| `provincies` | Provinces with INE codes and seat counts |
| `municipis` | Municipalities linked to provinces |
| `eleccions` | Election events with date metadata |
| `eleccions_municipis` | Per-municipality vote totals |
| `persones` | Candidate personal data |
| `candidatures` | Political parties and coalitions |
| `candidats` | Candidate–party–province assignments |
| `vots_candidatures_mun` | Votes per party per municipality |
| `vots_candidatures_prov` | Votes and seats won per party per province |
| `vots_candidatures_ca` | Votes per party per autonomous community |
